Locate the UI component, section, or screen described by the user.

Improve spacing and whitespace while preserving the existing layout and design style.

Rules:

* do not change content
* do not redesign the UI
* only improve spacing and layout balance

Improve the following aspects:

Element spacing:

* ensure consistent spacing between related elements
* ensure unrelated elements have larger separation
* maintain clear visual grouping

Padding:

* balance internal padding inside cards, containers, and panels
* ensure buttons and inputs have comfortable padding

Vertical rhythm:

* maintain consistent spacing between headings, text blocks, and controls
* ensure sections have clear breathing room

Consistency:

* avoid random spacing values
* use visually consistent spacing steps when possible
* maintain alignment with nearby elements

Important:

* preserve the design direction
* prioritize readability and clean layout
