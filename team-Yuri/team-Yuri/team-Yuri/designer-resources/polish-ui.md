Locate the relevant UI component, screen, or section described by the user.

Then polish the UI while preserving the existing design direction.

Rules:

* do not change layout structure unless needed for clarity
* do not change content or meaning
* do not introduce new visual styles unless requested

Improve the following aspects:

Spacing:

* ensure consistent spacing between related elements
* improve padding inside cards, buttons, inputs and containers
* maintain balanced vertical rhythm

Typography:

* ensure comfortable line-height for readability
* maintain clear hierarchy between headings, subheadings and body text
* avoid overly tight text blocks
* avoid excessive letter-spacing

Alignment:

* align related UI elements consistently
* ensure buttons, inputs and labels feel visually connected
* maintain visual grouping of related elements

Layout clarity:

* reduce visual clutter
* improve breathing room between sections
* keep the result clean, balanced and professional
