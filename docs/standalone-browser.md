# הרצת האפליקציה רק בדפדפן (בלי Node)

## קבצים

| קובץ | תיאור |
|------|--------|
| **`tennis-app-browser.html`** | העתק נוח — לחיצה כפולה ב-Explorer |
| **`run.html`** | אותו תוכן (שמירה על שם ישן) |

שניהם **זהים**: אפליקציית דמו עם `localStorage` — מאמן / שחקן, מערכת שבועית, הוספת שחקנים ושיעורים.

## איך לפתוח

1. **אפליקציה מלאה:** `npm.cmd run dev` ואז בדפדפן `http://localhost:3000` (אם הפורט תפוס — לעיתים `3001`; ראו פלט בטרמינל).  
2. **מפעיל מהיר:** לחיצה כפולה על `הפעלת-האפליקציה.html` — קישורים לשרת + לגרסת HTML.  
3. **רק דפדפן:** לחיצה כפולה על `tennis-app-browser.html` או `run.html`  
4. או: גרירה של הקובץ לחלון Chrome / Edge  

## מה *לא* כלול בגרסה הזו

- אין שרת — אין חיבור ל־API של Next.js  
- אין שיעורים חוזרים מלאים, חודש, LLM, תשלומים — אלה בפרויקט `npm run dev`  

## גיבוי נתונים מקומיים

הנתונים בדפדפן בלבד. לגיבוי: ייצוא ידני מ־DevTools → Application → Local Storage (או להשתמש באפליקציית Next עם `storage/data`).
